OCELOT

Welcome to the project documentation for OCELOT.

OCELOT integrates transcriptional regulation with genome-scale metabolic models to support two main tasks:

  • TF knockout simulation, optionally followed by essentiality evaluation
  • Metabolic engineering, including strategy search and post hoc validation
